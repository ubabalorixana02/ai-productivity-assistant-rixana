# AI Workplace Productivity Assistant

## Project Overview

The AI Workplace Productivity Assistant is a web-based application designed to improve workplace efficiency through artificial intelligence.

The platform helps professionals automate common workplace tasks, including email generation, meeting note summarisation, and task planning.

## Features

### Smart Email Generator

* Generates professional emails
* Supports multiple tones
* Editable output

### Meeting Notes Summarizer

* Summarises meeting discussions
* Extracts key decisions
* Identifies action items and deadlines

### AI Task Planner

* Prioritises tasks
* Creates schedules
* Provides productivity recommendations

## Technologies Used

* Lovable AI
* React
* TypeScript
* GitHub
* Supabase

## Responsible AI Statement

This application uses artificial intelligence to assist users with workplace productivity tasks. Users should review and verify all AI-generated content before professional use.
